1. The Objective

The main objective of this document is to provide a beginner-friendly introduction to Version Control Systems. It aims to teach you how to manage your code history locally using Git, share and collaborate on the cloud using GitHub, and navigate your computer using the Terminal.

2. What You Learn
Terminal Basics: How to interact with your computer using text commands like ls (list files), cd (change directory), mkdir (make folder), and touch (create file).

Local Git Workflow: The process of tracking code changes through commands such as git init (starting a project), git add (preparing files), and git commit (saving a snapshot).

Collaboration: How to connect your local work to the internet using git push to upload code and git pull to download updates from others.

Project History: Using git log to see a full timeline of every change made to the project.
